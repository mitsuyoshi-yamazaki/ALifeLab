# 人工生命の箱庭（仮
## 概要
プログラミングした生命を世界に放って生態系がどのように変化するか観察するゲーム

## プレイ方法
- [チュートリアル](./docs/tutorial.md)

## ゲームシステム
### 要件
- プログラミングでまったく自由な行動を行える
- 人工生命である
  - それ自体はOpen-Endedでない系に、人間の想像力を投入することでOpen-Endedを達成する

### 仕様
#### モデル
- 計算
  - Computorが最初
- 位置
  - 現状ではcollisionはない

#### 実装
- import順序：
  - primitive -> world_object -> module -> world等

## アイデア
- 自動でエネルギーを分配する循環系をつくることができる
- アップデートで抽象度を落としていく
